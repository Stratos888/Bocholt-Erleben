Veröffentliche deine Events im passenden monatlichen Paket. Zum Start klären wir Quelle, Paket und den passenden Veröffentlichungsweg kurz per Mail.

Als Veranstalter starten

Monatlich kündbare Pakete ab 9,99 € / Monat. Der Start läuft aktuell persönlich per Mail, die Abrechnung per Rechnung.

So läuft der Start

  1. Du meldest dich einmal per Mail.
  2. Wir klären Paket, offizielle Quelle und Rechnungsdaten.
  3. Danach legen wir den passenden Veröffentlichungsweg fest.
  4. Neue Events und Änderungen laufen anschließend gesammelt über diesen Weg.

Pakete

  • Starter 9,99 € / Monat · bis zu 3 Events
  • Aktiv 19,99 € / Monat · bis zu 8 Events
  • Dauerhaft 39,99 € / Monat · unbegrenzt

Alle Pakete sind monatlich kündbar. Welches Paket passt, klären wir beim Start kurz per Mail.

Quelle oder Vorlage

Neue Events und Änderungen laufen nach dem Start nicht als freie Einzelmails, sondern über einen festen Weg.

Bevorzugter Weg

  • Offizielle Website mit Eventseiten oder Veranstaltungskalender
  • Kalenderfeed oder andere maschinenlesbare Quelle
  • Einfache Standardvorlage, wenn keine Quelle vorhanden ist

Zum Start brauchen wir

  • Name des Veranstalters oder der Location
  • Ansprechpartner und Rechnungsdaten
  • Link zur offiziellen Website oder Eventquelle
  • Erste Events oder Link zum Kalender

Bevorzugt nutzen wir eine offizielle Quelle. Wenn das nicht möglich ist, arbeiten wir mit einer einfachen Standardvorlage.

Kurz & klar

  • Wie läuft die Veröffentlichung aktuell? Der Start wird persönlich per Mail eingerichtet. Danach stimmen wir einen festen Veröffentlichungsweg mit dir ab.
  • Muss ich jedes Event einzeln per Mail schicken? Nein. Bevorzugt nutzen wir eine offizielle Quelle. Alternativ arbeitest du mit einer einfachen Standardvorlage.
  • Ist das kostenlos? Nein. Veröffentlichungen laufen über monatlich kündbare Pakete ab 9,99 € pro Monat.
